== Description ==

InstaPic plugin used for fetched instagram images from username,hashtag or both.

A few notes about the sections above:

*  It show instagram images.
*  It show images from username and hasthtag.
*  Show images from short code.

   

== Installation ==
1. Downalod plugin from here and install in plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to wp-admin->InstaPic and use the settings to show images.



== Frequently Asked Questions ==

= Where will images showing =
